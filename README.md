# crypto-volatility-prediction
Cryptocurrency Volatility Prediction using Machine Learning
## 📌 Project Overview
This project focuses on predicting cryptocurrency price volatility using historical market data and machine learning techniques.  
The goal is to analyze price behavior, engineer meaningful features, and build predictive models to estimate volatility.

## 🔁 Pipeline Architecture
The project follows the below pipeline:

1. Data Collection  
2. Data Preprocessing  
3. Feature Engineering  
4. Exploratory Data Analysis (EDA)  
5. Model Training  
6. Model Evaluation  
7. Volatility Prediction
